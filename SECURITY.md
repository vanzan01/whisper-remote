# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please open an issue on GitHub or contact the maintainer directly.

## Supported Versions

| Version | Supported |
| ------- | --------- |
| latest  | Yes       |

## Security Considerations

This application:
- Runs a local HTTP server on your network
- Simulates keyboard input via Windows API
- Does not transmit data to external servers
- Requires your phone and PC to be on the same network

See [README.md](README.md) for more details.
